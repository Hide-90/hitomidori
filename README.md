# 人と緑の手づくり工房 記録メディア再資源化事業サイト

NPO法人「人と緑の手づくり工房」が取り組む、ビデオテープ・CD・DVDなどの記録メディア再資源化事業を紹介する、GitHub Pages向けの1ページWebサイトです。

## 構成

- `index.html` — サイト本文
- `style.css` — レスポンシブデザインとレイアウト
- `script.js` — モバイルメニュー、スクロール時のヘッダー表示、表示アニメーション
- `assets/flow.png` — 事業の概念図
- `.nojekyll` — GitHub PagesでJekyll処理を無効化

## 公開方法

1. このリポジトリをGitHubへプッシュします。
2. GitHubのリポジトリ設定から **Settings > Pages** を開きます。
3. 公開元として現在のブランチとルートディレクトリを選択します。
4. 表示されるURLへアクセスするとサイトを確認できます。

## 画像の差し替え

添付画像を利用する場合は、画像ファイルを `assets/flow.png` として配置してください。同じファイル名で差し替えるだけで、サイト内の概念図が更新されます。

## ローカル確認

ブラウザで `index.html` を直接開くか、以下のような簡易サーバーで確認できます。

```bash
python3 -m http.server 8000
```
